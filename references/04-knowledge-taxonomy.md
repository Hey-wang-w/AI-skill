# 知识分类体系与ID命名规范

> 本文件是知识点分类归属和ID分配的唯一依据。
> 新知识点入库前必须查阅本文件确定正确的归属和ID前缀。

---

## 术语前置解释（零基础必读）

| 术语 | 解释 |
|------|------|
| 分类体系（Taxonomy） | 按层级组织知识点的结构，类似电脑的文件夹树状结构，大文件夹套小文件夹 |
| L0/L1/L2/L3/L4 | Level的缩写，表示分类层级。L0是最大的阶段分类，L4是最细的章节分类 |
| ID | Identifier的缩写，每个知识点的唯一标识符，格式为6段式，如`1-AI-ML-DL-LLM-01` |
| 缩写 | 每个分类层级用英文大写缩写表示，如ML=Machine Learning（机器学习），DL=Deep Learning（深度学习） |
| 正交分类 | 正交（Orthogonal）指"互相独立、不重叠"。本系统ID只能表示树状层级，但一个知识点可能同时属于多个维度（如LLM既是NLP领域又是生成模型），这时候需要用"知识标签"来补充标记 |
| 知识标签 | 飞书多维表格中的多选字段，用于标记ID层级无法表达的跨维度归属，如`生成模型`、`基础模型`等 |
| 双维度分类 | 深度学习(DL)下的L4同时按"技术领域"（CV/NLP/其他）和"建模逻辑"（判别/生成）两个维度划分，ID中只取其一，另一维度通过知识标签体现 |

## 一、6级分类体系说明

严格按**概念包含关系**归属：大概念包含小概念，上级包含下级。

```
L0阶段 → L1领域 → L2模块 → L3主题 → L4章节 → 知识点（艾宾浩斯单元）
```

| 层级 | 说明 | 示例 |
|------|------|------|
| L0 | 学习阶段，共10个预置阶段 | 阶段1：AI通识认知 |
| L1 | 知识领域，阶段1统一为"人工智能" | 人工智能 |
| L2 | 知识模块，是L1下的大分支 | AI导论、搜索博弈、机器学习、AI伦理 |
| L3 | 知识主题，是L2下的主题分类 | 机器学习下的：概率统计基础、基础概念、经典模型、深度学习 |
| L4 | 知识章节，是L3下的具体知识点组 | 深度学习下的双维度分类：维度一（技术领域）神经网络基础、计算机视觉(CV)、序列模型/NLP(SEQ)；维度二（建模逻辑）判别模型、生成模型(GEN)；大语言模型(LLM)在ID中作为DL下L4，概念上属于生成模型→基础模型分支 |
| - | 知识点（KP） | 每个具体的艾宾浩斯记忆单元 |

⚠️ 关键原则（必须严格遵守概念包含关系）：
- 人工智能 **包含** 机器学习，机器学习 **包含** 深度学习（不是平级关系！）
- 概率统计是数学基础学科，但在阶段1（AI通识认知）中作为机器学习的前置知识归入ML模块下；未来阶段4（AI数学基础）会独立展开
- 跨学科知识点（如贝叶斯公式）在当前阶段1归入人工智能领域下的ML模块
- 用"知识标签"字段解决正交分类（如一个知识点可同时打"Transformer""生成模型"标签）

> - **神经网络基础(NN)** 是所有深度学习的公共基础，任何DL知识点都需要先掌握NN。
> - **维度一（技术领域视角）**：按应用领域划分，包括计算机视觉(CV)、自然语言处理/序列模型(NLP/SEQ)、其他专用网络（如GAN）。
> - **维度二（建模逻辑视角）**：按模型的建模方式划分，包括判别模型、生成模型(GEN)。
> - **生成模型(GEN)** 下包含：生成式AI定义(GEN-03)、幻觉(GEN-04)、GAN(GEN-02)、基础模型与预训练微调(GEN-05)、多模态(GEN-06)。
> - **LLM（大语言模型）** 虽然ID挂在DL下作为L4章节（即`1-AI-ML-DL-LLM-XX`），但概念上属于"生成模型→基础模型"分支。ID保持6段式不变，需通过**知识标签**标注其维度二归属：给LLM知识点加上`生成模型`、`基础模型`、`LLM`标签。
>
> **ID迁移说明**：飞书中原有旧ID `1-AI-ML-BASE-TYPE-01` 已更新为 `1-AI-ML-CORE-TYPE-01`，本文档统一使用新ID。

## 五、知识标签体系

知识标签是多选字段，用于解决ID层级无法表达的**跨分类正交问题**。

> **为什么需要知识标签？** ID是6段式树状结构（如`1-AI-ML-DL-LLM-01`），只能表达"LLM属于DL"这一条包含关系。但实际上LLM既是NLP领域技术，又是生成模型，还是基础模型——这些"多归属"关系无法塞进6段ID里，需要通过知识标签来补充标注。

**常用标签及使用场景**：

### 按技术领域
- `监督学习` `无监督学习` `强化学习`
- `深度学习` `神经网络` `Transformer` `CNN`
- `LLM` `大语言模型` `生成模型` `基础模型` `判别模型`
- `CV` `计算机视觉` `NLP` `自然语言处理`
- `概率` `统计` `贝叶斯`

### 按概念类型
- `核心概念` `算法原理` `历史背景`
- `伦理问题` `实践技巧` `常见误区`

### 按能力层级
- `需要理解` `需要记忆` `需要应用`

**双维度标注示例**：
- LLM知识点（ID为`1-AI-ML-DL-LLM-XX`）：需加标签 `生成模型`、`基础模型`、`LLM`，标注其属于"维度二（建模逻辑）→生成模型→基础模型"分支。
- GAN知识点（ID为`1-AI-ML-DL-GEN-02`）：本身就在GEN下，同时属于"其他网络"领域，无需额外标签。

新知识点入库时根据内容选择合适的标签，可新增标签但应保持一致性。
