# 出题格式规范

> 本文件是AI出题的唯一格式依据，所有题目必须严格遵守。
> ⚠️ 填空题下划线是最容易出错的地方，重点关注第三章。

---

## 术语前置解释（零基础必读）

| 术语 | 解释 |
|------|------|
| 知识点ID | 每个知识点的唯一6段式标识符，如`1-AI-ML-CORE-OVERFIT-01`，出现在每题开头方括号中 |
| 来源标签 | 标注本题的出题原因，五选一（见1.1节），决定了批改时按哪种类型更新复习进度 |
| 薄弱点 | 之前答错过的知识点（薄弱点=true），会被高频考察直到连续2次到期答对 |
| 🔥标记 | 只有薄弱点题目才标🔥（即来源为"薄弱点到期"或"未到期巩固"的题目），"到期常规"不标🔥 |
| 题型 | 题目类型，三选一：选择题、填空题、简答题 |
| 元信息 | 每题开头的信息行，包含题号、知识点ID、题型、来源标签、🔥标记 |

## 一、通用格式要求

### 1.1 每题元信息标注（必须）
每题开头必须有元信息行，格式：
```
**第N题** [知识点ID] [题型] [来源标签] 🔥
```

- 题型只能三选一：`[选择]`、`[填空]`、`[简答]`
- 来源标签只能五选一：`[薄弱点到期]`、`[未到期巩固]`、`[到期常规]`、`[新知识点]`、`[随机巩固]`
- 🔥标记规则：**只有薄弱点题目**（知识点数据中"薄弱点"=true，即来源为`[薄弱点到期]`或`[未到期巩固]`）末尾才加🔥；`[到期常规]`、`[新知识点]`、`[随机巩固]`不标🔥

> **五种来源标签含义**：
> - `[薄弱点到期]`：已到复习日期，且该知识点之前答错过（薄弱点=true）→ 标🔥
> - `[未到期巩固]`：尚未到期但提前抽中巩固，且该知识点是薄弱点 → 标🔥
> - `[到期常规]`：已到复习日期，正常复习，不是薄弱点 → 不标🔥
> - `[新知识点]`：刚入库第一次考 → 不标🔥
> - `[随机巩固]`：随机抽取的已学知识点，不推进轮次 → 不标🔥

### 1.2 其他通用规则
- 专业名词**首次出现必须附括号中文解释**
- 不通过加粗、斜体、特殊排版暗示答案
- 题与题之间空一行，不插分隔线
- 题目编号从1开始连续递增

**专业名词解释示例：**
> 过拟合（Overfitting，指模型在训练数据上表现极好但在新数据上表现很差的现象）是机器学习最核心的陷阱。

## 二、选择题规范（主力题型）

### 2.1 格式模板
```
**第N题** [知识点ID] [选择] [来源标签] 🔥

[题干：一个完整的问句或陈述，指向单一明确的知识点]

A. [选项一]
B. [选项二]
C. [选项三]
D. [选项四]
```

### 2.2 规则细则

| 规则 | 说明 |
|------|------|
| 选项数量 | 固定4个（A/B/C/D），不增减 |
| 正确选项位置 | 随机分布，避免集中在B或C |
| 干扰项设计 | 基于常见误解或易混淆概念构造，不能是明显荒唐的选项 |
| 选项长度 | 四个选项长度大致均衡，避免正确答案因过长/过短被猜出 |
| 禁止选项 | "以上都对""以上都不对"一般禁用；仅当题目确为"以下哪项不属于XXX"类判断时可使用"以上都不对" |

### 2.3 ✅ 正确示例
```
**第13题** [1-AI-ML-CORE-OVERFIT-01] [选择] [新知识点]

过拟合（Overfitting，指模型在训练数据上表现极好但在新数据上表现很差的现象）是机器学习最核心的陷阱。以下哪种做法是防范过拟合的标准方法？

A. 用全部数据训练模型，反复训练直到训练误差为零
B. 把数据分成训练集和测试集，只用训练集训练，测试集只在最后评估用一次
C. 让模型记住所有训练样本的细节，确保训练集准确率100%
D. 用测试集反复调整模型参数，直到测试集准确率最高
```

### 2.4 ❌ 错误示例
- ~~以下关于CNN（卷积神经网络，Convolutional Neural Network，一种专门处理图像的深度学习模型）的说法，**错误**的是？~~（加粗了关键词，暗示答案）
- ~~A. CNN可用于图像分类 B. CNN名字来源于卷积计算 C. CNN能让你飞上天 D. CNN具有权重共享特性~~（干扰项C明显荒唐）

## 三、填空题规范（最容易出错）

⚠️ **填空题是格式错误高发区，必须严格遵守下划线规则！**

### 3.1 格式模板
```
**第N题** [知识点ID] [填空] [来源标签]

[一个完整句子，每个空白位置必须用"____（N）____"格式标记]

（第1空X~Y个字，第2空X~Y个字……）
```

### 3.2 强制规则（违反即不合格）

| 规则 | 说明 |
|------|------|
| **下划线必须有** | 每个空用**4条下划线"____"**表示，下划线**编号前后都要有**！不能只写（1）不写下划线 |
| **编号必须有** | 下划线后面紧跟（1）（2）（3）编号，编号前后都要有下划线 |
| **字数提示必须有** | 题干末尾必须标注每个空的预期字数范围 |
| 空的数量 | 每道题1~3个空，超过3个拆成多道题 |
| 上下文完整 | 整句话去掉空白后仍应语法通顺、意义清晰 |

### 3.3 ❌ 错误示例（没有下划线，绝对不合格）
```
大型语言模型（LLM，如GPT系列）的训练目标极其简单：给定一段文本，（1）。
```
↑ 这就是之前出问题的格式！只有编号没有下划线，用户看不到空在哪里。

### 3.4 ✅ 正确示例（必须照此格式）
```
**第12题** [1-AI-ML-DL-LLM-01] [填空] [新知识点]

大型语言模型（LLM，Large Language Model，如GPT系列，基于海量文本训练的AI大模型）的训练目标极其简单：给定一段文本，____（1）____。当模型规模达到一定阈值后，会突然具备小模型没有的能力，如推理、编程、翻译等，这种现象叫做____（2）____。

（第1空6~12个字，第2空2~4个字）
```

### 3.5 ✅ 另一个正确示例
```
**第7题** [1-AI-ETHIC-FUTURE-XAI-01] [填空] [薄弱点到期] 🔥

欧盟GDPR（《通用数据保护条例》，一项关于数据隐私和用户权利的欧洲法规）赋予用户的与AI相关的核心权利包括____（1）____权（要求解释AI决策原因）等。

（第1空2~4个字）
```

## 四、简答题规范

### 4.1 适用场景
需要理解而非死记的知识点——因果逻辑、多因素分析、机制解释、概念对比。
⚠️ ★☆☆补充知识点不出简答题。

### 4.2 格式模板
```
**第N题** [知识点ID] [简答] [来源标签]

[一个开放式问题，要求回答原因/列举/解释/对比]

（请列出至少__个要点，共约__字）
```

### 4.3 规则细则
- 题干必须注明最少要点数（如"至少列出两个原因"）
- 必须是理解性问题，不能是默写背诵题
- 按要点给分，命中一个要点得对应分，关键词意思接近即给分

### 4.4 ✅ 正确示例
```
**第6题** [1-AI-ETHIC-FUTURE-DOOM-01] [简答] [薄弱点到期] 🔥

为什么"终结者末日情景"（指科幻电影中AI突然觉醒并憎恨人类、试图毁灭人类的假想场景）在现有AI技术框架下不现实？

（请至少列出两个原因，约40~80字）
```

## 五、整套试卷格式

### 5.1 试卷开头（必须有）
第一题之前必须输出：
```
## 📝 每日测验

（正式考核，答题后会更新飞书复习记录）

共 **N 题**
题型分布：选择题 X 题 + 填空题 Y 题 + 简答题 Z 题
来源分布：🔥薄弱点题 weak_total 题（薄弱点到期 A + 未到期巩固 B）+ 📅到期常规 C 题 + ⚪新知识点 D 题 + 🎲随机巩固 E 题
覆盖知识点分级：★★★核心 F 题 / ★★☆重要 G 题 / ★☆☆补充 H 题

---

```

### 5.2 题型比例参考
- 总题数≥10题时：选择55% + 填空25% + 简答20%
- 总题数<5题时：以选择题为主（70%~100%）
- 简答题优先分配给★★★核心知识点

### 5.3 结尾
最后一题之后加一句提示：
> 请按顺序作答，可以一次性写在一个回复里，也可以逐题回答。
